전체 삭제

전체 삭제 방법 1

gg // 첫번째 줄로 이동
dG // 현재 위치 이하 모두 삭제 (d + Shift g)
전체 삭제 방법 2 (비주얼 라인 모드)

gg // 첫번째 줄로 이동
VG // shift v(비주얼 라인 모드), shift g(현재 위치 이하 모두 선택) (shift v + shift g)
d  // 삭제
