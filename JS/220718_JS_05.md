### 연습문제2

- 다음과 같이 동작하도록 findAge 함수를 구현하세요
- **hint**. string, 객체 메소드 복습

```jsx
const arr = [{ 
  name: 'kim', 
  age: 10, 
}, { 
  name: 'park', 
  age: 15, 
}, { 
  name: 'lee', 
  age: 12, 
}, { 
  name: 'choi', 
  age: 13, 
}, { 
  name: 'jin', 
  age: 20, 
}, { 
  name: 'woo', 
  age: 30, 
}]; 

findAge(' jIN'); // 20; 
findAge(' wOO   '); // 30
```

- **클린 코드:**
    
    ```jsx
    const arr = [{ 
      name: 'kim', 
      age: 10, 
    }, { 
      name: 'park', 
      age: 15, 
    }, { 
      name: 'lee', 
      age: 12, 
    }, { 
      name: 'choi', 
      age: 13, 
    }, { 
      name: 'jin', 
      age: 20, 
    }, { 
      name: 'woo', 
      age: 30, 
    }]; 
    
    const findAge = (name) => {
    	const formattedName = name.toFlowerCase().trim();	
    	const user = arr.find(item => item.name === formattedName);
    	return user.age;
    }	
    
    findAge(' JIN');
    ```
